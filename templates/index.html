Bagaimana Cara Kerjanya?

1. Input Data Peminjam

2. Ekstraksi Fitur

Sistem mengolah data menjadi 22 fitur numerik terstandarisasi

3. Cari Cluster Terdekat

Hitung distance ke centroid dan temukan cluster paling mirip

4. Prediksi Hasil

Estimasi kelayakan berdasarkan default rate cluster

Analisis dan Visualisasi Data

Metode Elbow - Penentuan Jumlah Cluster Optimal

Metode Elbow digunakan untuk menentukan jumlah cluster optimal dengan melihat penurunan inertia. Dari grafik di bawah, terlihat bahwa k=4 adalah titik optimal (elbow).

Visualisasi Cluster 2D dengan PCA

Dengan menggunakan Principal Component Analysis (PCA), data 22-dimensi direduksi menjadi 2 dimensi untuk visualisasi. Warna berbeda mewakili cluster berbeda.

Profil Cluster - Fitur Kunci

Perbandingan nilai rata-rata untuk fitur penting di setiap cluster. Ini membantu memahami karakteristik unik dari setiap segmen peminjam.

Heatmap - Semua Fitur Cluster

Heatmap komprehensif yang menampilkan rata-rata semua fitur numerik untuk setiap cluster. Warna lebih terang menunjukkan nilai lebih tinggi.
